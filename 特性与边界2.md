

## 系统特性与解决方案边界





| 要素 | 内容 |
| --- | :--- |
| 问题ID | P2 |
| 目标 | 用户使用此系统能够知道90%新学期所需书籍 |
| 解决方案 | 为用户提供一个检索课程要用书籍的平台，用户可以直接搜索课程关键字，或者根据分类索引发现自己可能需要的书籍 |







---







### ◆ 系统特性







| 编号 | 内容 |
| --- | :--- |
| 1 | 通过搜索书籍关键字可以检索到相关书籍 |
| 2 | 通过检索目录可以找到目标分类，从而找到相关书籍 |
| 3 | 书目的管理由第三方书店负责 |
